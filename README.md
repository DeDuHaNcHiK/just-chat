Welcome just-chat
=================

Install
-------
    $ virtualenv --no-site-packages flask
    $ ./flask/bin/easy_install pip
    $ ./flask/bin/pip install -r ./requirements.txt

Run
---
    $ source ./flask/bin/activate
    (flask):$ python run.py
